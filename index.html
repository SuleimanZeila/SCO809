Course Details

This course covers various software engineering techniques and methodologies, including but not limited to requirements analysis, design principles, testing strategies, and project management methodologies.

Git & GitHub

Git is a distributed version control system that allows collaboration between developers working on the same project. GitHub is a web-based platform that hosts Git repositories and provides additional collaboration features such as issue tracking, pull requests, and project management tools.

Common Git Commands

  • git init
  • git add
  • git commit
  • git push
  • git pull
